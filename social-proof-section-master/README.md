Overview

▪️The Social Proof Section is a responsive webpage that displays customer ratings and testimonials. The project is designed to look clean, modern, and visually appealing while maintaining a responsive layout across desktop and mobile devices.

Features

▪️Responsive design for desktop and mobile screens
▪️Customer rating section with five-star reviews
▪️Testimonial cards with customer images and feedback
▪️Clean layout created using Flexbox
▪️Consistent color scheme using CSS variables
▪️Google Fonts integration for better typography

Built With

▪️HTML5
▪️CSS
▪️Flexbox
▪️CSS Variables
▪️Media Queries
▪️Google Fonts (League Spartan)

What I Learned

Working on this project improved my understanding of responsive web design and Flexbox. I practiced organizing HTML into meaningful sections and learned how to position elements more efficiently using CSS. I also became more knowledgeable using media queries to adjust layouts for different screen sizes and using CSS variables to keep the stylesheet organized.
And also I got a lot of new understandings on the flexbox layout. Generally I got a lot of knowledge on Website recreation and also creation using HTML and CSS.

Challenges

One of the biggest challenges was creating a layout that matched the design while keeping it responsive. Positioning the rating boxes and testimonial cards required careful use of Flexbox and spacing.On this section I mean on the flexbox and spacing I struggled a lot to get the needed output. I also learned the importance of testing different screen sizes to ensure the design remained consistent.
